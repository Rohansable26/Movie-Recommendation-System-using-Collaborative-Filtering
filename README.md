# Recommender System

This project implements a basic movie recommender system using Collaborative Filtering and Cosine Similarity.

## Features
- User similarity calculation
- Movie recommendations
- Collaborative filtering

## Technologies Used
- Python
- Pandas
- Scikit-learn

## Run Project

Install dependencies:

pip install -r requirements.txt

Run:

python recommender.py
